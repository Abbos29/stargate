Integrate Stargate now

Embed the Stargate Transfer module in any application created with React/NextJS, giving users instant ability to swap native assets cross-chain, accessing Stargate’s unified liquidity pools
Effortless WidgetGreat User Experience

First, install the package from either npm or yarn:
> npm i @layerzerolabs/stargate-ui
> yarn add @layerzerolabs/stargate-ui
And then freely use it inside your app:
import React, {FC} from 'react';
import Widget, {themes} from '@layerzerolabs/stargate-ui';

const Home: FC = () => {
  return <Widget theme={themes.dark} />;
};
